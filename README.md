# spotify-behavioral-analytics
End-to-end analytics project using Python, MySQL, SQL, and Power BI to transform Spotify streaming history into actionable behavioral insights.

# Spotify Behavioral Analytics Dashboard

## Overview

This project analyzes Spotify Extended Streaming History to understand user engagement, listening behavior, and preference trends through an end-to-end analytics pipeline.

The project uses Python for data preparation, MySQL for relational data storage and SQL analysis, and Power BI for interactive dashboards.

## Technology Stack

- Python (Google Colab)
- Pandas
- MySQL
- SQL
- Power BI
- GitHub
